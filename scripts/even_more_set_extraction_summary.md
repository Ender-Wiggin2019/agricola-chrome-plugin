## 提取方法
1. 使用`srt_to_txt.py`脚本将SRT字幕文件转换为纯文本
2. 通过自然语言理解识别卡牌名称和描述边界
3. 根据`.cursor/convert.mdc`中定义的JSON格式进行结构化输出
4. 每张卡牌包含：name（名称）、tier（梯度）、no（编号-待补充）、desc（描述）

## 注意事项
1. 视频是口述评价，因此描述内容较为口语化
2. 卡牌编号（no字段）需要后续补充，因为字幕中未明确提及
3. 部分卡牌的梯度信息在视频中有提及，已标注在tier字段
4. 描述内容尽量保留原视频中的评价和分析
