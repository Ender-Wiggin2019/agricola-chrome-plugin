# 功能和设计说明

## 核心功能实现

### 1. 搜索框组件 (`SearchBox.tsx`)
- **特性**:
  - 带有放大镜图标的现代化搜索框
  - 实时搜索，无需点击按钮
  - 使用 useCallback 优化性能
- **设计**: 使用 shadcn/ui 的 Input 组件，居中布局，最大宽度 2xl

### 2. 卡牌结果组件 (`CardResult.tsx`)
- **特性**:
  - 显示卡牌编号、中文名、英文名
  - 三个评级系统（Baitu、EN、Chen）各自独立显示
  - 每个评级下方显示对应的评论和评论者信息
  - 统计数据独立展示
- **设计**: 使用 Card 组件，带有悬停效果和阴影

### 3. Tier 徽章组件 (`TierBadge.tsx`)
- **特性**:
  - 动态颜色编码
  - 显示 "+" 表示有额外描述
- **设计**: 使用 Badge 组件，颜色根据评级自动计算

### 4. 统计徽章组件 (`StatsBadge.tsx`)
- **特性**:
  - 圆形徽章显示 ADP 值
  - 颜色表示受欢迎程度
- **设计**: 圆形设计，32x32px，白色文字

### 5. 统计详情组件 (`StatsDetails.tsx`)
- **特性**:
  - 显示完整的统计数据（PWR、ADP、APR、Draw Play Rate）
  - 根据数值自动着色
- **设计**: 两列布局，标签在左，数值在右

## 性能优化策略

1. **React 优化**
   - 使用 `useMemo` 缓存搜索结果
   - 使用 `useCallback` 避免不必要的重新渲染
   - 搜索结果限制为 3 个，减少 DOM 节点

2. **构建优化**
   - Vite 快速热更新
   - Tailwind CSS 生产环境自动 purge
   - TypeScript 类型检查

3. **加载优化**
   - 数据文件按需加载
   - 组件懒加载就绪（如需要可扩展）

## 用户体验设计

### 交互设计
- **即时反馈**: 输入即搜索，无需等待
- **清晰的视觉层级**:
  - 标题使用粗体和主色调
  - 描述使用浅色文字
  - 徽章使用醒目的颜色编码
- **响应式布局**: 适配各种屏幕尺寸

### 信息架构
```
搜索框
  ↓
搜索结果列表
  ↓
单个卡牌
  ├── 基本信息（编号、名称）
  ├── Baitu 评级 + 评论
  ├── EN 评级 + 评论
  ├── Chen 评级 + 评论
  └── 统计数据
```

### 视觉反馈
- 卡片悬停时显示阴影效果
- 输入框聚焦时显示边框高亮
- 加载状态显示友好提示
- 无结果时显示明确的提示信息

## 可扩展性

### 已实现的可扩展功能

2. **多评级系统**: 可轻松添加更多评级来源
3. **统计数据**: 支持任意数量的统计指标

### 未来可扩展方向

2. **卡牌对比**: 并排对比多张卡牌
3. **收藏功能**: 保存喜欢的卡牌
4. **导出功能**: 导出搜索结果或卡牌列表
5. **暗黑模式**: 已预留 Tailwind 暗黑模式配置
6. **多语言**: 类型定义已支持多语言
